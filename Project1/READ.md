# Project1 - Storm Viewer

XCode 10 - Swift 4

-----

This application is created to demonstrate the knowledge of:

- UIKit
- XCode StoryBoards & Auto Layout
- Table & Image views
- File Manager
- View Controllers
- UIImage
- UIBarButtonItem
- UIActivityViewController
