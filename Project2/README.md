# Project2 - Flag Game

XCode 10 - Swift 4

-----

This application is created to demonstrate the knowledge of:

- UIKit
- XCode StoryBoards & Auto Layout
- Interface Builder
- Asset Catalogs
- UIButton
- CALayer
- UIColor
- UIAlertController
